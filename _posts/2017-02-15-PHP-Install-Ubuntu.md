## Ubuntu apt 安装 PHP7

```shell
$ sudo apt install php7.0
```


### 安装扩展包

```shell
$ sudo apt install php7.0-curl php7.0-gd php7.0-json php7.0-zip php7.0-xml \
php7.0-ldap php7.0-mbstring php7.0-mcrypt php7.0-snmp php7.0-mysql
```
